## Project Description
A responsive weather dashboard that displays current weather conditions, 5-day forecasts, and 3-hour interval forecasts using the OpenWeather API. The project focuses on API integration, data transformation, and UI structuring for real-world frontend applications. DOM Manipulation codes

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeather API

## Features
- City-based weather search
- Current weather display (temperature, humidity, wind, pressure)
- 5-day aggregated forecast
- 3-hour interval forecast
- Skeleton loading UI (ready state)
- Error handling for failed API requests
- MapLibre integration hook (future enhancement ready)

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
   OR
   Replace API key in `app.js`
3. Enter a city name and search

## Lessons Learned
- Working with external REST APIs
- Transforming time-series weather data into meaningful UI components
- Handling asynchronous JavaScript using async/await and Promise.all
- Structuring frontend logic for scalability

## Challenges Faced
- Parsing 3-hour interval weather data into a 5-day summary
- Managing asynchronous API calls efficiently
- Designing a layout that accommodates multiple forecast types in a single view
